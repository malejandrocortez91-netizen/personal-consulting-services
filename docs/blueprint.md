# **App Name**: Consultant Canvas

## Core Features:

- Interactive Resume Timeline: Displays work experience and education in a dynamic, chronological timeline. Users can click on entries to view detailed descriptions.
- Service Descriptions: Details the services offered, including operations management, process improvement, and talent development. Incorporates client testimonials where applicable.
- Project Highlights Section: Showcases key projects and accomplishments with metrics, descriptions, and visuals (if available).
- Integrated Contact Form: A simple, effective contact form for potential clients to inquire about services. Automatically sends inquiries to the consultant's email.
- Skills Summarizer: Uses provided text (resume/CV) to summarize key skills and experiences to dynamically populate the skills section.
- Chatbot with Notifications and Auto-Response: A chatbot that sends an email notification to malejandro.cortez91@gmail.com when someone wants to hire or get services. It also sends an automated message with the PDF version of the resume to the user.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3148) for a professional and trustworthy feel.
- Background color: Light gray (#E0E0E0) for a clean and modern backdrop.
- Accent color: A muted blue-gray (#778899) to provide subtle contrast and highlight key elements.
- Body font: 'Inter', a sans-serif font providing a neutral and readable appearance, suitable for body text.
- Headline font: 'Space Grotesk', a sans-serif font for headlines that offers a computerized feel.
- Use simple, professional icons to represent skills and services. Icons should be in the muted blue-gray accent color.
- Clean, well-spaced layout with clear sections for resume, services, projects, and contact information.
